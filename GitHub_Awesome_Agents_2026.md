# قائمة مستودعات GitHub العالمية للوكلاء والمهارات 2026

هذه القائمة تحتوي على مئات الموارد المجمعة من أقوى مستودعات GitHub المتخصصة في الذكاء الاصطناعي والوكلاء.

---

## 1. أطر عمل الأوركسترا (Orchestration Frameworks)
| المشروع | الوصف | اللغة |
| :--- | :--- | :--- |
| [CrewAI](https://github.com/crewAIInc/crewAI) | إطار عمل لفرق الوكلاء المتعددة، معتمد من آلاف الشركات. | Python |
| [LangGraph](https://github.com/langchain-ai/langgraph) | بناء وكلاء يعتمدون على الرسوم البيانية والحالات (Stateful). | Python |
| [AutoGen](https://github.com/microsoft/autogen) | إطار عمل من مايكروسوفت للوكلاء المتعددين المعتمدين على الأحداث. | Python |
| [Mastra](https://github.com/mastra-ai/mastra) | إطار عمل TypeScript متكامل مع دعم RAG و MCP. | TypeScript |
| [PydanticAI](https://github.com/pydantic/pydantic-ai) | إطار عمل يركز على سلامة الأنواع (Type-safe). | Python |
| [Smolagents](https://github.com/huggingface/smolagents) | وكلاء يكتبون وينفذون كود Python مباشرة من Hugging Face. | Python |

## 4. أدوات وبنية تحتية (Agent Tooling)
*   **MCP (Model Context Protocol)**: بروتوكول جديد من Anthropic لربط الوكلاء بالأدوات.
*   **AgentOps**: لمراقبة وتتبع أداء الوكلاء في الإنتاج.
*   **OpenClaw**: نظام تشغيل للوكلاء.
